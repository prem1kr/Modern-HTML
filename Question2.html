HTML entities are special codes used to represent reserved characters and symbols in HTML. Here are five commonly used HTML entities:

1. `lt` - represents the less-than symbol ("<").

2. `gt` - represents the greater-than symbol (">").

3. 'amp' - represents the ampersand symbol ("&").

4. 'quot' - represents the double quotation mark (").

5. 'apos' - represents the single quotation mark (apostrophe) ('), though it's less commonly used.

These entities are used to ensure that certain characters are correctly displayed in HTML and do not interfere with the markup structure.
